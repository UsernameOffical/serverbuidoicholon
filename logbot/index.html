Features

Log Bot is designed to log stuff for you in a channel. Good for administration purposes.

- Logs deleted messages
- Logs who joins and leaves the server
- !ping Command to display ping.
- !botinfo Command to display the bots information .
- !uptime Command to display how long the bot has been online.
- !prefix Displays the guilds current prefix.
- !setprefix <prefix> Sets the guilds prefix. Default !.
- !setchannel <#channel-name> Sets the guilds logging channel. Default logs.
- ~LogBotMasterReset~ master reset command (must have admin perms)
