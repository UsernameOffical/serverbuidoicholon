Introduction

This Wiki has all my scripts ace permissions and setting them up. Use the sidebar and find the script of which your trying to set Ace Permissions up for.

Need further support? Join my Discord here.

Area of Patrol (AOP)

Simply copy and pate the below into your server.cfg and replace the HEX-ID-HERE with your steam hexadecimal. A hexadecimal looks like this; 110000000000000(With different characters).
add_ace Fax.AOP faxes.aopcmds allow

add_principal identifier.steam:HEX-ID-HERE Fax.AOP

Server Notification

Simply copy and pate the below into your server.cfg and replace the HEX-ID-HERE with your steam hexadecimal. A hexadecimal looks like this; 110000000000000(With different characters).
add_ace group.admin fax allow               # allow all commands for FAXES Script admins 
add_ace group.moderator fax allow           # allow all commands for FAXES Script moderators
add_ace group.moderator fax.cmds deny	    # disallow FAXES Script Perms for moderators

add_principal identifier.steam:HEX-ID-HERE group.admin
